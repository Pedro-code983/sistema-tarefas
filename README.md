# 📌 Sistema de Gerenciamento de Tarefas

Uma aplicação desenvolvida para ajudar na organização diária, permitindo o controle completo de tarefas e pendências de forma intuitiva.

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Python / JavaScript / Java
- **Framework/Biblioteca:** Flask / React / Node.js
- **Banco de Dados:** SQLite / MySQL / MongoDB
- **Estilização:** CSS3 / Tailwind CSS

## 🎯 Funcionalidades
- [x] Criação de novas tarefas com título e descrição.
- [x] Marcação de tarefas como "concluídas".
- [x] Edição e exclusão de tarefas existentes.
- [ ] *Filtro por categorias ou prioridades (Em breve)*.
