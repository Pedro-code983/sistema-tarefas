# 🚀 Roysse Tasks v2.0 - Gestão de Alta Performance gamificada

O **Roysse Tasks v2.0** é uma plataforma web de gerenciamento de tarefas que utiliza conceitos de **gamificação** para aumentar a produtividade. Em vez de uma lista de afazeres comum, o sistema transforma tarefas diárias em "Missões Ativas" que, ao serem cumpridas, geram recompensas em formato de "Cookies" (moeda interna do sistema) para serem trocadas em uma "Loja de Mimos".

## 🎯 Diferenciais do Projeto (Gamificação)
- **Sistema de Missões:** Tarefas divididas por prioridade (Média, Alta, etc.) e categorias.
- **Economia Interna (Cookies):** O usuário acumula pontos de acordo com a dificuldade da missão concluída.
- **Análise de Produtividade:** Um gráfico interativo que exibe o balanceamento entre missões concluídas e pendentes.
- **Loja de Mimos:** Uma área integrada para troca de recompensas obtidas através do esforço diário.
